## Nymphaea Tool에 대하여
- 게임 기획자의 업무에 있어 데이터 작업에 할애하는 시간을 줄이고 Game Design적인 업무에 집중하고자 개발한 엑셀 자동화 도구입니다.
- 빈도가 높은 작업의 단축 기능, 데이터의 유효성을 검증하는 보조 기능, 빈도는 낮지만 많은 시간이 소요되는 작업의 자동화 기능 등을 지원하는 것을 목표로 하고 있습니다.


## 파일 정보
### ThisAddIn
- 툴 자체 기능에 대한 포괄적인 기능을 실행하는 곳입니다.

- 리본, 버튼 등 상호 작용 가능한 UI에 대한 기능을 실행하는 곳입니다.

## 사용법
### Insert Image
- 많은 양의 이미지를 자동으로 셀 크기에 맞게 삽입해주는 기능.
> - File Path : 파일 경로를 입력.
> - Extention : 파일의 확장자를 입력. 입력하지 않을 시, 자동으로 png파일로 세팅.
> - File Name Col : 파일 이름 데이터를 가지고 있는 컬럼의 시작 셀 주소를 입력. 연속된 데이터에 대해서만 체크.
> - Insert Col : 이미지를 삽입할 컬럼의 시작 셀 주소를 입력.
> - Insert : 이미지 삽입 기능 실행.
> - Delete : 이미지 삭제 기능 실행.
